java双列集合学习

发表于2024-11-13|更新于2024-11-21
|字数总计:645|阅读时长:2分钟|阅读量:

Map

特点：

  1. 双列集合一次要存储一对数据，分别为键和值
  2. 键不能重复，值可以重复
  3. 键和值是一一对应的，每一个键只能找到自己对应的值
  4. 键＋值的整体，称之为“键值对”或者“键值对对象”，在java中叫做“Entry对象”

Map方法

  • 创建Map集合的对象
  • 添加元素
  • 移除所有元素
  • 判断是否包含指定的键
  • 判断是否包含指定的值
  • 判断集合是否为空
  • 集合的长度，键值对的个数

键值对

  • 迭代器遍历
  • 增强for遍历
  • lambda表达式

HashMap

  • HashMap底层是哈希表结构
  • 依赖hashCode方法和equals方法保证键的唯一
  • 如果键存储的是自定义对象，需要重写hashCode和equals方法，如果值是自定义对象，则不需要
